# Mini_Shopping_Web
这是一个微信小程序商城，作为个人学习小程序的练习作品。
包括首页、搜索页、商品页、购物车等电商网站常见页面，具有登录、搜索等常见功能。

## 技术栈
* Uniapp、Vue3、Vuex；

## 开发
请使用HbuilderX进行开发
包安装：
```sh
npm install
```
## 目录结构

```
Mini_Shopping_Web
├─ .hbuilderx          // HbuilderX 配置
├─ App.vue             // 入口页面
├─ LICENSE             // 许可
├─ README.md           // 说明文档
├─ components          // 组件
├─ index.html          // 首页入口文件
├─ main.js             // 入口文件
├─ manifest.json       // 小程序配置文件
├─ mixins              // mixin文件
├─ package-lock.json   // 项目模块信息
├─ package.json        // 项目模块配置
├─ pages               // 页面
├─ pages.json          // 页面路由配置
├─ static              // 静态资源
├─ store               // store
├─ subpkg              // 分包
├─ uni.scss            // 公共css
├─ uni_modules         // uniapp组件
└─ utilies             // 全局公用方法

```
